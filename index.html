Introduction

Lilac API documentation.

API Token Authentication

The Lilac API provides an API token based authentication system. To authenticate using a valid token, include your token under the HTTP header X-API-KEY. How to Find Your Token

  1. Login Lilac dashboard https://app.v3.dev.amili.asia/
  2. Go to API Key Management

  1. Click Create if you haven't generated your API Key yet

  1. Copy API Key
